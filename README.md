# supermemory-exploration

Sandbox for exploring [Supermemory](https://supermemory.ai) — memory infrastructure for AI apps.

Primary docs entrypoint: [https://supermemory.ai/docs/intro](https://supermemory.ai/docs/intro)

## What this repo is for

Hands-on exploration of Supermemory product surface, APIs, SDKs, and integration patterns:

- Core concepts (memory, users, containers, search)
- REST API and SDKs
- Ingestion, retrieval, and memory lifecycle
- Agent / LLM integration patterns
- Comparison notes vs alternatives (when relevant)
